Crie um formulário de cadastro para um site de e-commerce. O formulário deve conter os seguintes campos:

    Nome completo (campo de texto)
    Endereço de e-mail (campo de e-mail)
    Senha (campo de senha)
    Selecione seu país (menu dropdown)
    Gênero (botões de rádio)
    Aceitar os termos de serviço (checkbox)

Estilize o formulário para que:

    O botão de envio fique destacado ao passar o mouse (efeito hover).
    O formulário tenha uma largura de 400px e esteja centralizado na tela.

Dicas:

    Utilize <form>, <input>, <select> e <textarea>.
    Use margin, padding e hover para melhorar a estilização.
